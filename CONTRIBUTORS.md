## 🎯 How to Become a Contributor

We welcome contributions from everyone! See [CONTRIBUTING.md](docs/CONTRIBUTING.md) for guidelines.

### Ways to Contribute

1. **Code** - Fix bugs, add features, improve performance
2. **Documentation** - Write docs, improve guides, fix typos
3. **Design** - Create UI mockups, design icons, improve UX
4. **Testing** - Find bugs, report issues, test new features
5. **Translation** - Translate UI and documentation
6. **Support** - Help other users, answer questions
7. **Feedback** - Provide suggestions, report issues, give feedback

# 🚀 Quick Start Guide - Get Started in 5 Minutes

## ⚡ Fastest Way to Use AI Clipper

### Step 1: Download & Install (2 minutes)

1. Download installer: `AI-Clipper-v1.0.0-x64-setup.exe`
   - From: [GitHub Releases](https://github.com/YOUR_USERNAME/ai-clipper/releases)

2. Double-click installer
3. Click "Next" through the wizard
4. Wait for installation to complete
5. Launch "AI Clipper" from Start Menu

### Step 2: Get API Key (2 minutes)

1. Visit: https://makersuite.google.com/app/apikey
2. Sign in with Google account
3. Click "Create API key"
4. Copy the API key
5. Open AI Clipper and paste the key
6. Click "Save" (one-time setup)

### Step 3: Generate Your First Clip (1 minute)

1. Paste YouTube URL (example: `https://www.youtube.com/watch?v=xxxxx`)
2. Click "Generate Clips"
3. Wait 5-10 minutes for AI processing
4. Preview generated clips
5. Select clips to export
6. Click "Export All"
7. Upload to TikTok/Shorts/Reels!

**Done!** 🎉

## ⚠️ Troubleshooting

### Issue: "Invalid URL"

**Solution:**
- Make sure it's a valid YouTube URL
- Try using the share link from YouTube
- Example: `https://www.youtube.com/watch?v=xxxxx`

### Issue: "API Key Required"

**Solution:**
- Get free key from: https://makersuite.google.com/app/apikey
- Paste in application settings
- Click "Save"

### Issue: "Download Failed"

**Solution:**
- Check internet connection
- Try a different video
- Some videos may be restricted

### Issue: "Export Failed"

**Solution:**
- Check disk space
- Close other applications
- Try exporting one clip at a time

### Issue: "Processing Too Slow"

**Normal:** 5-10 minutes per video
**Faster:** Better CPU, more RAM
**Optimal:** 16 GB RAM, SSD storage
